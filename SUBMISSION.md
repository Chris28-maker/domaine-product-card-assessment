## Implementation Summary

This product card was built from scratch inside a headed Shopify theme environment using Liquid, TailwindCSS utility classes, and vanilla JavaScript.

It supports:

- Sale badge and markdown pricing
- Variant color swatches
- Alternate color imagery on swatch click
- Secondary image on hover for the selected variant
- Product title, vendor/brand name, and pricing display

## Technical Notes

TailwindCSS is loaded through the CDN snippet for a Shopify Admin based prototype. In a production theme, Tailwind should be compiled into a static CSS asset.
