# QA Checklist

Use this before submitting the assessment.

## Visual QA

- [ ] Card matches the Figma layout as closely as possible.
- [ ] Image container has rounded corners and a thin border.
- [ ] Sale badge appears as a red outlined pill.
- [ ] Swatches are circular and aligned below the product image.
- [ ] Selected swatch has a visible ring.
- [ ] Brand name appears above the product title.
- [ ] Product title is styled as a blue link.
- [ ] Compare-at price has a strikethrough.
- [ ] Sale price is red.

## Functional QA

- [ ] Product card loads inside a Shopify theme page.
- [ ] Clicking a color swatch changes the product image.
- [ ] Hovering over the product image shows the selected variant's secondary image.
- [ ] Product card link updates to the selected variant URL.
- [ ] Sale badge hides when the selected variant is not on sale.
- [ ] Compare-at price hides when the selected variant is not on sale.
- [ ] Product title, vendor, and price are visible.
- [ ] Card works on mobile width.

## Submission QA

- [ ] GitHub repository is public or accessible to the reviewer.
- [ ] README explains setup and implementation choices.
- [ ] Working Shopify preview/live link is available.
- [ ] Product media alt text has been set correctly.
